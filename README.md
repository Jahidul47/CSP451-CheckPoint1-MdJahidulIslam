# CSP451 CheckPoint 1

## Description

This project demonstrates the basics of Git, GitHub, version control, Markdown documentation, and a simple static website using HTML, CSS, and JavaScript.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Jahidul47/CSP451-CheckPoint1-MdJahidulIslam.git
```

2. Open the project folder:

```bash
cd CSP451-CheckPoint1-MdJahidulIslam
```

## Usage

Open `index.html` in a web browser to view the website.

Useful Git commands:

```bash
git status
git log --oneline --graph --all --decorate
git branch
```

## Contributing

This project uses a feature-branch workflow. New work should be done on a separate branch and then merged into `main` using a pull request.

Commit messages should follow Conventional Commits format:

```text
feat: add new feature
fix: correct a problem
docs: update documentation
style: improve formatting
refactor: reorganize code without changing behaviour
```
